# Album Web Service

A RESTful web service built with Go and the Gin framework for managing a collection of music albums.

## Description

This project is a simple web service that provides a RESTful API for managing album records. It demonstrates basic CRUD operations (Create, Read) using the Gin web framework, a popular HTTP web framework written in Go.

## Features

- **Get all albums**: Retrieve the complete list of albums
- **Get album by ID**: Retrieve a specific album by its ID
- **Add new album**: Create a new album record
